基本视图

我们定义一种组成APP界面的最基本的元素：View，所有的APP界面都由各种View组成。
所以我们在网页的最外层包裹一个.view-list作为最基础的视图

<div class="view-list">
    <div class="view">...</div>
    <div class="view">...</div>
</div>

线性布局

该布局由上到下按顺序堆叠

<div class="view-list">
    <div class="view">...</div>
    <div class="view">...</div>
</div>

CSS：

.view { display: block; }
